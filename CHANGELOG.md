# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Initial release of Waste Classification System
- YOLOv11 integration with Ultralytics
- 10 Jupyter notebooks covering full ML pipeline
- NumPy implementations of core ML concepts:
  - Convolution operations
  - Activation functions (ReLU, SiLU, Sigmoid)
  - Pooling operations
  - Detection metrics (IoU, NMS, mAP)
  - Loss functions (BCE, Focal, CIoU)
- Streamlit web application for deployment
- Comprehensive documentation:
  - README.md with project overview
  - CODE-THEORY.md linking code to theory
  - SYSTEM_DESIGN.md with architecture diagrams
- Test suite for math implementations
- YOLO format dataset conversion tools

### Technical Details
- Python 3.9+ compatible
- PyTorch 2.0+ backend
- Ultralytics YOLOv11 8.0+
- Streamlit 1.28+ for web interface
