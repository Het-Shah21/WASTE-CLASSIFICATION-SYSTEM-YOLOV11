# Assets Directory

This directory contains images, diagrams, and other visual assets for documentation.

## Contents

| File | Description |
|------|-------------|
| `banner.png` | Project banner for README |
| `backbone_stages.png` | CNN backbone visualization |
| `neck_architecture.png` | FPN + PANet diagram |
| `yolo_grid.png` | Multi-scale detection grid |
| `model_comparison.png` | YOLOv11 variants comparison |
| `bce_loss.png` | BCE loss visualization |
| `focal_vs_bce.png` | Focal vs BCE comparison |
| `iou_visualization.png` | IoU concept diagram |
| `nms_demo.png` | Non-Maximum Suppression demo |
| `pr_curve.png` | Precision-Recall curve |
| `training_results.png` | Training metrics |
| `model_predictions.png` | Sample predictions |

## Generating Assets

Assets are generated by running the Jupyter notebooks:
- `05_cnn_fundamentals.ipynb`
- `06_detection_metrics.ipynb`
- `07_yolo_architecture.ipynb`
- `08_loss_functions.ipynb`
- `10_model_training.ipynb`
